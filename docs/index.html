Running the code

KAUST

  1. Run an the MKL script to load the required MKL envrioment variables into our shell: ``` sh source /opt/intel/oneapi/mkl/latest/env/vars.sh intel64 ``

  2. Add the magma library to theld` linked environment path variable:

    ``` sh export LD_LIBRARY_PATH=<path to library>/magma-2.5.4/lib:$LD_LIBRARY_PATH ```

  3. Set the local MAGMA root directory path (needed for make.inc file):
    export MAGMA_DIR=/home/x_pollakgr/applications
  1. Run the makefile: ``` sh make ```
